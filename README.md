# catalogo-filmes-previsao-tempo

## Dependencias
**rodar:**

`dotnet add package Npgsql.EntityFrameworkCore.PostgreSQL --version 9.0.4`

`dotnet add package TMDbLib`

## Observações

- criar banco de dados `catalogo_filmes` em postgres

- script para criar tabela Filme esta em: `catalogo-filmes-previsao-tempo/Data/db.sql`

- atualizar dados de conexão no arquivo `catalogo-filmes-previsao-tempo/appsettings.json`
